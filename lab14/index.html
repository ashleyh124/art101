    Challenge

    The challenge of the lab is to create a function that depends on conditionals.

    Problems

    The problems we encountered during the lab was the CSS originally creating two borders around the results. We found out that it was because of our previous CSS code, but when we changed it to a comment, it worked and only created one border.

    Results

    We created a button that says "Submit" with a user input box. We added an event listener and JS events, so that when the user types in their name and click the button, it will take the length of their name and sort them into a Hogwarts house. Within the function, there is a condition that sorts the user into one of the four houses depending on the remainder of their name divided by 4.
